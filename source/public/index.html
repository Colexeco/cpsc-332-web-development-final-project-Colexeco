CPSC 332 Final Project Presentation

Project Management App

Evan Shoemaker

I made a very simple project managment app. Users can create projects with titles, descriptions, deadlines, mark them as completed, and assign tasks to them. They can view all of their created projects in one page. They can also edit those projects to reflect progress.

Functional Requirements:

  1. Users must be able to log in and out of the application using a username and password (front)

  2. Users must be able to log back in with previous name and password from database (back)

  3. Users must be able to create project/task entries (front)

  4. Users must be able to retrieve previously created project/task entries (back)

  5. Users must be able to add deadlines to projects/tasks (front)

  6. Users must be able to retrieve previously created deadlines of projects/tasks (back)

  7. Users must be able fill out descriptions for projects/tasks (front)

  8. Users must be able to retrieve previously created descriptions for projects/tasks (back)

  9. Users must be able to modify projects/tasks to reflect progress (front)

  10. Users must be able to retrieve past modifications to view progress (back)
